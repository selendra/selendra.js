# @selendra

This library provides additional typing information for user to access Selendra by using [polkadot.js](https://github.com/polkadot-js/api).

# Getting Started
- Install rush package manger
```bash
npm install -g @microsoft/rush
```

- Build evm

```bash
yarn run build:evm
```
- Run evm
```bash
yarn run evm
```
